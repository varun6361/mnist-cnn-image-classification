# mnist-cnn-image-classification
# 🧠 Handwritten Digit Classification using CNN

## 📌 Project Overview

This project implements a **Convolutional Neural Network (CNN)** using TensorFlow to classify handwritten digits (0–9).
The model is trained on the **MNIST dataset** and achieves high accuracy.

---

## 🚀 Features

* Image classification using deep learning
* CNN architecture implementation
* Model training and evaluation
* Visualization of results
* Real-time prediction demo

---

## 🧰 Technologies Used

* Python
* TensorFlow / Keras
* NumPy
* Matplotlib

---

## 📊 Dataset

* MNIST Dataset (handwritten digits)
* 60,000 training images
* 10,000 testing images

---

## 🧠 Model Architecture

* Conv2D + ReLU
* MaxPooling
* Conv2D + ReLU
* MaxPooling
* Flatten
* Dense (64 neurons)
* Output Layer (Softmax)

---

## 📈 Results

* Achieved **~98% accuracy** on test data
* Model successfully predicts handwritten digits
